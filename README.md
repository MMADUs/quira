# Quira

Version: `0.1.0`

![Quira](asset/quira.png)

## About

Quira is a lightweight C++ quantum computing library for building and simulating
quantum circuits to explore quantum information.

The project is designed to keep the core API small, portable, and easy to embed
inside larger applications while still feeling familiar to users of modern
quantum computing libraries.
